## 介绍
ENGLISH-LIKED 是一个基于php构建的英语单词听写系统，可通过 PHP 个性化单词数量单词内容，配置单元，配置页面。
## 部署
无需执行任何控制台操作，只需要2步骤<br>
### 1. 下载仓库文件<br>
建议通过Git Clone下载，当然你也可以用zip解压。
```bash
git clone https://github.com/CHEN-EXE/ENGLISH-LIKED.git
```
